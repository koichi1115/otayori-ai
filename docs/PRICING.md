# おたよりAI 料金体系案

## 方針

おたよりAIはユーザーが自身のAPIキーを使ってAI解析を行うため、AI利用コスト自体はユーザー負担。アプリの収益はプレミアム機能への課金で得る。

---

## 案A: フリーミアム + サブスクリプション（推奨）

### Free プラン（無料）
| 機能 | 制限 |
|------|------|
| PDF/画像のAI解析 | 月10回まで |
| 解析結果の閲覧・編集 | 無制限 |
| 履歴の検索・フィルタ | 無制限 |
| 子供・施設の登録 | 子供2人まで |
| AIモデル選択 | 全モデル利用可 |
| Google連携 | なし |
| LINE通知 | なし |
| Google Drive同期 | なし |

### Premium プラン（月額 ¥480 / 年額 ¥3,800）
| 機能 | 内容 |
|------|------|
| PDF/画像のAI解析 | **無制限** |
| 子供・施設の登録 | **無制限** |
| Googleカレンダー連携 | **利用可** |
| Googleタスク連携 | **利用可** |
| Google Drive同期 | **利用可** |
| LINE通知 | **利用可** |
| 優先サポート | メール対応 |

### 価格設定の根拠
- **月額¥480**: コーヒー1杯分。子育て世帯にとって払いやすい価格帯
- **年額¥3,800**: 月額の約66%（年間で¥1,960お得）。継続率向上が目的
- 競合のプリゼロは月額¥300だがAI解析なし。AI解析+Google連携の価値で¥480は妥当
- 月10回無料は「試してもらう」に十分（月のプリント枚数は平均5〜15枚程度）

---

## 案B: 買い切り

| プラン | 価格 | 内容 |
|--------|------|------|
| 無料版 | ¥0 | 月5回解析、子供1人まで |
| 完全版 | ¥1,480 | 全機能無制限 |

**メリット**: ユーザーにわかりやすい、サブスク疲れの層に刺さる
**デメリット**: 継続収益がない、アップデートのモチベーション維持が難しい

---

## 案C: 解析回数課金

| パック | 価格 | 回数 |
|--------|------|------|
| 無料 | ¥0 | 月5回 |
| 50回パック | ¥250 | 50回（有効期限なし） |
| 200回パック | ¥800 | 200回（有効期限なし） |

**メリット**: 使った分だけ、心理的ハードルが低い
**デメリット**: Google連携等の課金ポイントが曖昧

---

## 推奨: 案A（フリーミアム + サブスク）

**理由**:
1. 無料で十分に使えてレビューが集まりやすい
2. Google連携・LINE通知は「あると便利→ないと困る」に変わるため、自然にアップグレードされやすい
3. App Store的にもサブスクモデルが推奨されている（Auto-Renewable Subscription）
4. 年額プランで LTV を伸ばせる

---

## 実装上の考慮

| 項目 | 方法 |
|------|------|
| 課金基盤 | `expo-in-app-purchases` or `react-native-iap` |
| 無料回数カウント | SQLite settings テーブルに `monthlyAnalysisCount` + `lastResetMonth` |
| Premium判定 | App Store レシート検証（サーバー不要、ローカルレシート検証可） |
| 復元ボタン | 設定画面に「購入を復元」ボタン必須（審査要件） |
| トライアル | 初月無料トライアル（App Store Connect で設定） |
